Thanks for downloading PhotoManager for Linux!

To install, "cd" into this directory, run "chmod +x install.sh", and then run "sudo ./install.sh" file.

Note: you may have to logout and back in again to see PhotoManager in your applications.

To uninstall, "cd" into this directory, run "chmod +x uninstall.sh", and then run the "sudo ./uninstall.sh" file.

If you don't want to install the program but just want to run it, you can "chmod +x photomanager" and then "./photomanager" to run it where the binary is.
